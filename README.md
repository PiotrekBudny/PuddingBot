# PuddingBot
PuddingBot - discord bot for private use.
Requires a '.env' file with "DISCORD_TOKEN" value set.

COMMANDS:

'help'

ex. of usage: $$help

'purge_messages_from'

remove all messages sent by user in current channel. Executable by Admins only. 

ex. of usage: $$purge_messages_from @user
